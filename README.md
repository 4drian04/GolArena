# ⚽ LigaFan

LigaFan es una aplicación Android diseñada para los aficionados al fútbol español. Mantente al día con las últimas noticias de La Liga, demuestra tus conocimientos deportivos en divertidos trivials y colecciona a tus jugadores favoritos mientras amplías tu colección de cartas.

## 📱 Características

### 📰 Noticias de la Liga Española

Consulta las últimas novedades, resultados, fichajes y acontecimientos relacionados con la liga española de fútbol desde una interfaz sencilla e intuitiva.

### 🎯 Trivials de Fútbol y Deportes

Pon a prueba tus conocimientos respondiendo preguntas sobre:

- Fútbol nacional e internacional
- Historia del deporte
- Deportes en general

Completa trivials para obtener recompensas y seguir progresando dentro de la aplicación.

### 🎁 Sistema de Sobres

Gana sobres al superar trivials y desafíos.  
Los sobres pueden contener cartas de distintos jugadores con diferentes niveles de rareza.

### 🃏 Colección de Jugadores

Construye tu propia colección de futbolistas:

- Consigue jugadores actuales e históricos
- Descubre cartas raras y especiales
- Completa tu colección

### 🤖 Chatbot Deportivo

La aplicación incluye un chatbot capaz de responder preguntas relacionadas con el fútbol y los deportes.

> **Nota:** El chatbot es experimental y puede ofrecer respuestas inexactas o incompletas.

## 🚀 Instalación

```bash
git clone https://github.com/4drian04/GolArena.git
```

1. Abre el proyecto en Android Studio
2. Sincroniza Gradle
3. Ejecuta la aplicación en un emulador o dispositivo Android

---

## 🛠️ Tecnologías utilizadas

- Kotlin
- Android Studio
- Android Jetpack
- Retrofit
- Room Database
- Material Design
- API de noticias deportivas
- Chatbot basado en IA (experimental)

---

## 🎮 Cómo jugar

1. Lee las noticias de la liga española
2. Completa trivials deportivos
3. Gana sobres como recompensa
4. Abre sobres para conseguir jugadores
5. Mejora y gestiona tu colección
